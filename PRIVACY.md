# Privacy Policy

**MD Feedback** does not collect, store, transmit, or process any personal data.

## VS Code Extension

- Runs entirely on your local machine
- All annotations and review data are stored as local files in your workspace
- No telemetry, analytics, or usage tracking of any kind
- No network requests are made by the extension

## MCP Server

- Communicates exclusively via stdio (standard input/output) with the host AI tool
- No network connections are initiated
- No data is sent to external services
- All file operations are confined to the workspace directory you specify

## Third-Party Services

MD Feedback does not integrate with any third-party analytics, advertising, or data-processing services.

## Contact

If you have questions about this policy, please open an issue at:
