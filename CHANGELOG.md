## [0.9.5] - 2026-02-16

첫 안정 릴리스를 위한 버그 수정 및 품질 개선.

### Added
- AI 응답이 리뷰 패널에 직접 표시됩니다
- AI 응답 위에 다시 어노테이션을 달 수 있습니다

### Fixed
- 첫 번째 메모가 저장되지 않고 사라지는 문제
- MCP 서버 수정 후 어노테이션이 인식되지 않는 문제
- 세션 핸드오프 시 체크포인트가 유실되는 문제
- 동일 텍스트가 여러 번 등장할 때 어노테이션 위치가 부정확한 문제

### Improved
- 메모 포맷 호환성 통합
- 테스트 커버리지 확대

## [0.9.4] - 2026-02-15

초기 공개 릴리스.

### Added
- VS Code Extension: 마크다운 어노테이션 UI (Fix/Question/Highlight)

- 11개 AI 코딩 도구 Export 지원
- 키보드 단축키 1-2-3
